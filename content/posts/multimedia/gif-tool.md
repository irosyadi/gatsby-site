# GIF Tools

## GIF Editor

## GIF-Search Engine
1. Giphy, owned by Facebook
2. GIF Search via Google
`https://www.google.com/search?q=?&tbm=isch&tbs=itp:animated`

2. [Giftxt](https://giftxt.herokuapp.com/generate?text=), change behind `text=` with your text
